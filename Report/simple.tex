\begin{abstract}
Social networks like Twitter produce enourmeous amounts user-generated data that is a great source for data-mining algorithms to extract information about rising trends and opinions. We have designed an algorithm for identifying trending topics in twitter using the Misra-Gries algorithm. With the K-minimum values algorithm we can determine if a trending topics popularity is caused by many distinct mentions or is a result of a high number of retweets. We explain how we adapt the Twitter data stream to accomodate the mentioned algorithms.
\end{abstract}
